Android 022 使用 View Animation 動畫效果

By Azyueki

發表於 2015-04-14

文章目錄
  1. 1. 動畫類型
  2. 2. 建立方式

動畫類型

  • View Animation：指定動畫開始與結束時的物件屬性（透明度、大小、角度與動畫時間長度等）讓 Android 系統自動產生動畫過程中所有畫面。API Guide
  • Drawable Animation：類似卡通動畫的製作過程，自己指定每一影格播放的影像檔與時間長度，Android 系統依照我們的設定播放動畫。

建立方式

  • res 動畫資源檔（xml）
  • 程式碼中建立動畫物件並設定

動畫效果

  • Alpha：改變影像透明度，alpha 值 1 表示完全不透明，0 代表完全透明。
  • Scale：改變影像大小，scale 值 0 表示完全看不到，1 表示原影像大小，可獨立設定 x 和 y 方向的 scale。
  • Translate：改變影像位置，以 x 和 y 方向移動量表示。
  • Rotate：改變影像的角度。
